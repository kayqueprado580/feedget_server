# FeedGet - BFF

Sistema destinado a ser Widget para Feedback do seu sistema.

## Funcionalidades

* Criar um feedback.
* Enviar email para o responsável do sistema.

## Tipo Aplicação

 * BFF - API em NodeJS

## Stack

* Javascript
* Typescript
* Nodejs
* Express
* Prisma
* Nodemailer
* Jest

## Dependências

* PostgreSQL: Banco de dados responsável por armazenar todos os dados do feedback.

## Rodando a aplicação

````bash
npm run dev
````

## Rodando os testes

````bash
npm run test
````
